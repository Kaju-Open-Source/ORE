# ORE — Kaju Open Source

[![License: MIT](https://img.shields.io/badge/license-MIT-green)](LICENSE)

ORE turns academic documents into a shared, searchable, AI-powered
knowledge repository. **Three teams**, three folders:

```text
ore/
├── ai-rag/       🤖 AI team        — Python + Ollama
├── backend/      🧠 Backend team   — HTTP API (coming soon)
├── frontend/     🎨 Frontend team  — Next.js 16 UI (coming soon)
├── docs/         📚 shared docs
├── CONTRIBUTING.md    ← team rules
├── COMMANDS.md        ← every shell command (no `make`)
├── AGENTS.md          ← project context (for future you / AI assistants)
├── LICENSE
└── README.md          ← you are here
```

---

## I just cloned — what now?

1. Read [`CONTRIBUTING.md`](CONTRIBUTING.md) — pick your team.
2. Open **your team's folder** and follow its README.
3. To run tests / lint / dev server: [`COMMANDS.md`](COMMANDS.md).

| You are on…         | Open this folder        |
|---------------------|-------------------------|
| 🤖 **AI team**       | [`ai-rag/`](ai-rag/)    |
| 🧠 **Backend team**  | [`backend/`](backend/)  |
| 🎨 **Frontend team** | [`frontend/`](frontend/)|
