# 🎨 Frontend team — `frontend/`

A Next.js 16 + React 19 + Tailwind v4 + TypeScript app.

## Run it

See [`../COMMANDS.md`](../COMMANDS.md) § "Frontend team" — `npm install`,
`npm run dev`, etc.

## Boundaries

- You are the **only** team that edits this folder.
- Talk to the Backend **only** through the HTTP routes in the API
  contract (lives under `docs/`, not created yet).
- **Never import from another team's folder.**
